# BuildProtect

## General
Protect your builds with this plugin

## Commands
|Permissions|
|-----------|
|build.protect.wand|

## Features
- Create areas
- Edit areas
- Delete areas
- Comes with a wand for selections
- Custom Wand Item
- Custom Wand Name

## Help
- Break a block with the wand to select the first position
- Tap/click a block with the wand to select the second position
